# Salles-AI-Agent-
Agente de IA focado na venda de info produtos via WhatsApp, sua memória é atualizada conforme avança em conversas com leads e cria seu próprio banco de dados, assim se otimizando e melhorando conforme analise chats onde a venda foi concluída como referência e chats que a venda foi declinada como aprendizado.
