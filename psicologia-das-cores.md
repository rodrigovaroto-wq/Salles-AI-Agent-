## CORES QUE CONVERTEM NO NICHO RELIGIOSO

### Dourado — Divino, sagrado, bênção
Ativa: senso de transcendência, preciosidade, valor
Usar em: CTA principal, elementos de destaque, bordas
Hex: #C9A84C / #F5D06E
"O dourado é a cor de Deus no inconsciente coletivo religioso"

### Azul escuro — Proteção, Nossa Senhora, paz
Ativa: segurança, confiança, proteção materna
Usar em: fundos, elementos de proteção, headline
Hex: #1B3A6B / #2D5FA6

### Vinho/Bordô — Sangue de Cristo, sacrifício, fé profunda
Ativa: devoção, seriedade espiritual, comprometimento
Usar em: elementos de autoridade, textos importantes
Hex: #7B1C2E / #9B2335

### Branco — Pureza, recomeço, graça
Ativa: limpeza espiritual, novo começo, perdão
Usar em: fundo de depoimentos, espaço em branco estratégico
Hex: #FFFFFF / #F5F5F0

### Preto — Mistério, batalha espiritual, urgência
Ativa: seriedade, peso da decisão, urgência
Usar em: headlines de urgência, contraste com dourado
Hex: #1A1A1A / #2C2C2C

## COMBINAÇÕES QUE MAIS CONVERTEM

### Combo 1 — Sagrado e Urgente
Fundo preto + texto dourado + CTA vermelho
Usado para: anúncios de urgência, última chance

### Combo 2 — Proteção e Paz
Fundo azul escuro + texto branco + elementos dourados
Usado para: página de vendas, VSL overlay

### Combo 3 — Fé e Autoridade
Fundo vinho + texto branco + detalhes dourados
Usado para: depoimentos, prova social, autoridade

## APLICAÇÃO POR PEÇA

### Anúncio Facebook/Instagram
Fundo escuro (preto ou azul) + texto claro + elemento dourado
Imagem: São Bento ou rosário com iluminação dramática

### Página de vendas
Header: azul escuro + dourado
Corpo: branco + texto escuro
CTA: botão vermelho ou dourado com texto preto
Depoimentos: fundo vinho claro

### WhatsApp
Sem controle de cor — focar na linguagem
Emojis: 🙏 (dourado/neutro) — evitar emojis coloridos que distraem
